# Test Resources

This folder contains sample files for testing content sharing functionality.

## Recommended Test Files

Add the following types of files for comprehensive testing:

### Images
- `sample-chart.png` - Test image scaling with charts/diagrams
- `sample-photo.jpg` - Test photo display
- `sample-wide.png` - Test ultra-wide image (21:9 aspect ratio)
- `sample-tall.jpg` - Test tall/portrait image

### Documents
- `sample.pdf` - Multi-page PDF document
- `sample-single.pdf` - Single page PDF
- `sample.pptx` - PowerPoint presentation
- `sample.docx` - Word document

### Videos
- `sample-video.mp4` - Short video clip (recommended: 10-30 seconds)
- `sample-wide.mp4` - Wide aspect ratio video

### Audio
- `sample-audio.mp3` - Audio file for testing audio player

## Usage

1. Add your test files to this folder
2. When testing content sharing in the app, select these files
3. Verify:
   - Images scale properly and fill the available space
   - PDFs open in the iframe viewer
   - Videos play with proper controls and sizing
   - Audio files display with player controls

## File Size Recommendations

- Images: Under 5MB
- Documents: Under 10MB
- Videos: Under 50MB
- Audio: Under 10MB

Keep file sizes reasonable for smooth testing experience.
